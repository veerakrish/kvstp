# How to Download Temple Images

## Quick Guide to Get Images

### Option 1: Download from ourtemples.info (Recommended)

1. **Visit the temple page:**
   - Go to: https://ourtemples.info/temple/sri-venkateswaraswamy-temple-juvvalapalem-road-kallakuru-andhra-pradesh-534237/

2. **Download images:**
   - Right-click on any image
   - Select "Save image as..." or "Save picture as..."
   - Save to this `images` folder with the correct filename

3. **Required image names:**
   - `hero-bg.jpg` - Main temple exterior (landscape, wide image)
   - `temple-main.jpg` - Temple building view
   - `gallery-1.jpg` - Temple exterior
   - `gallery-2.jpg` - Main deity (Lord Venkateswara)
   - `gallery-3.jpg` - Temple interior
   - `gallery-4.jpg` - Prasadam (Pulihora Popu)
   - `gallery-5.jpg` - Festival or event
   - `gallery-6.jpg` - Devotees or temple scenes
   - `history-temple.jpg` - Any temple view (optional)

### Option 2: Use Google Images

1. Search for: "Kallakuru Venkateswara Swamy Temple"
2. Filter by "Large" size images
3. Download and rename to match the required filenames

### Option 3: Take Your Own Photos

If you have photos of the temple:
1. Resize them to recommended sizes (see below)
2. Save as JPG format
3. Use the exact filenames listed above

## Image Size Recommendations

- **hero-bg.jpg**: 1920x1080px (or larger, landscape orientation)
- **temple-main.jpg**: 800x600px or larger
- **gallery images**: 800x600px or larger (all same size recommended)
- **history-temple.jpg**: 800x600px or larger

## Image Format

- **Format**: JPG or JPEG (preferred) or PNG
- **Quality**: High quality (at least 800px width for gallery images)
- **File names**: Must be EXACTLY as listed (case-sensitive, no spaces)

## Quick Test

After adding images, open `index.html` in your browser to see them displayed.

## Note

The website will show placeholder images if the actual images are not found, so you can test the website even without images first.
