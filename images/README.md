# Images Folder

Please add temple images to this folder.

## Required Images:

1. **hero-bg.jpg** - Hero section background image (recommended: 1920x1080px or larger, landscape)
2. **temple-main.jpg** - Main temple image for the About section (recommended: 500x400px or larger)
3. **history-temple.jpg** - Image for the History section (optional, recommended: 800x400px or larger)
4. **gallery-1.jpg** - Temple exterior view
5. **gallery-2.jpg** - Main deity image
6. **gallery-3.jpg** - Temple interior
7. **gallery-4.jpg** - Prasadam (Pulihora Popu)
8. **gallery-5.jpg** - Festival/event image
9. **gallery-6.jpg** - Devotees or other temple scenes

## Image Sources:

Download images from:
- https://ourtemples.info/temple/sri-venkateswaraswamy-temple-juvvalapalem-road-kallakuru-andhra-pradesh-534237/

## Image Guidelines:

- **Format**: JPG or PNG
- **Size**: Recommended 800x600px or larger for best quality
- **File names**: Use exact names as listed above (case-sensitive)
- **Quality**: Use high-quality images for best display

## Hero Background Image:

The hero section background image can be updated in `index.html` (line ~38) by changing the CSS background-image URL.
